{app_name} - 智能启动器

启动应用的两种方式

1. 自动安装模式（推荐）

此启动器能够自动完成以下步骤：

  • 检测系统环境
  • 如果未安装Python，尝试自动下载并安装
  • 创建独立的Python虚拟环境
  • 安装所需的所有依赖
  • 启动Streamlit应用并打开浏览器

根据您的操作系统，请选择：

Windows启动器 Mac/Linux启动器
注意： 首次运行时需要下载和安装依赖，可能需要几分钟时间。
Windows用户： 请右键以管理员身份运行启动器。
Mac/Linux用户： 在终端中运行前，您可能需要先添加执行权限：
chmod +x 启动应用.sh

2. 手动安装模式（高级用户）

如果您熟悉命令行操作，也可以按以下步骤手动运行：

  1. 确保已安装Python 3.7+
  2. 创建虚拟环境： 
    python -m venv venv
  3. 激活虚拟环境： 
    Windows: venv\Scripts\activate
Mac/Linux: source venv/bin/activate
  4. 安装依赖： 
    pip install -r requirements.txt
  5. 启动应用： 
    streamlit run app.py

应用说明

本应用使用Streamlit构建，是一个交互式Web应用。成功启动后，您的默认浏览器会自动打开应用界面。

如果浏览器没有自动打开，请手动访问: http://localhost:8501
